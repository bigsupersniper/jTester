$context

$context 为全局注入对象，其最主要的属性是 params , 而 params 有以下属性

  • controller -- (string) controller名称
  • action -- (string) action名称
  • data -- (object) 参数集合

file (jTester.file)

file 有2个公用方法 openFile 和 saveFile

  • openFile($context) -- 用来上传的文件
  • saveFile($context) -- 用来下载的文件

http (jTester.http)

http 为http请求类 , 其构造方法为: http($context) , 有以下公用方法

  • get() -- http get请求
  • post() -- http post请求
  • put() -- http put请求
  • delete() -- http delete请求
  • down() -- 下载文件请求，已经封装在 file.saveFile($context)中了
  • upload() -- 上传文件请求，已经封装在 file.openFile($context)中了
